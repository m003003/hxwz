光是人类生活中的最重要元素之一。光不仅仅给人类带来光明，白天的阳光，夜间的月光和星光，给人类的行动指明了方向， 而且滋润着地球上万物的繁衍生长，包括人类自身，都需要光的能量以及它在生物体内产生的光合效应。 如果没有光，人类根本就不会诞生。
虽然自人类来到世间，就时时刻刻离不开光的照耀，但对光速和其本质的探讨，研究和测量，还是在现代科学建立以后的事。光速值的测量精度随着科学技术的进步越来越高，而对光速精度和认识的提高后又反馈到科学技术的各个领域中，促进它们进一步的发展。光速的测量史是科学进步史的重要一环。
在宇宙学的探讨和研究中，科学家就用光年，即光一年的行程， 来描写那超出常人想象力的浩渺无边的星际尺度。
何为速度?  速度是人类测量物体运动快慢的精确度量。 而要精确测量速度，必须先精确计量空间位置和时间。所以空间和时间是速度的两个基本元素。
相对于人类肉眼能够观察到的运动，如骏马奔驰，老牛拉车，雄鹰飞翔，等等，根本不能与光的运动速度相提并论，以致于在十七世纪以前，人们一直认为光速是无限的， 包括当时著名的科学家笛卡尔(René Descartes)也这样认为。中文中也有急如闪电这样的成语来描述光的超级速度。
就这样，时光流逝到十七世纪中叶的1632年，另一个著名科学家伽利略(Galileo Galilei)才开始认真地怀疑光速无限的传统假定， 并提出一个按现代眼光看来是超级简陋的方法来测量光速。有些人把他的这种测量方法称之为灯笼(Laterne)测量法。但根据现在流传下来的图片，这种意大利灯笼看起来并不像中国传统的灯笼，倒像是上个世纪七八十年代中国农村经常使用的煤油灯一样。当时，他也意识到光速是极高的，所以采用了在当时看来也算是标新立异的测试方式。他和他的助手分别带上一个灯笼和用于观察灯光的望远镜，各自站在从那个年代看来是相距很远至2到3英里的两个山头上。而灯笼被罩在壁上开有小孔的笼子里。为什么要用望远镜呢?  因为当时灯笼发出的光很弱，即使在黑夜里，传播2或3英里后，也需要借助望远镜才能用肉眼直接观察到那微弱的灯光。
当他把灯笼的灯罩打开后，灯笼里的光就会通过望远镜传到他助手的眼里，这时，他的助手立即打开自己的灯笼，让光通过伽利略的望远镜传到眼中。他通过测量时间差就能进一步确定光速的大小。可想而知，这样简陋的方法是不会成功的。 因为这样的距离下，人的反应时间和操作灯笼的时间和都远远大于光在这样短的距离下的传播时间，况且那时计量时间的仪表的误差也可能大于这段传播时间，所以，尽管他发表了他的测量过程，却没有留下任何可靠的测量数据。
回顾十七世纪测量光速的起始，我们现在可能会暗自发笑。伽利略那么伟大的科学家，在那个年代都要用那么原始的设备来测量速度。而现在的我们, 可以用肉眼看不见的原子运动来测量光速。这足以说明，在四百年的时间跨度中，科学技术给我们各种活动带来了翻天覆地的变化。
时光又推移了几十年。终于在1676年，丹麦物理学家罗默 (Ole Römer) 发现光速有限并且成功地测量了它的值。罗默在德语里的意思是罗马人。看来，他的祖先也是意大利人，和伽利略有着相同的祖先，所以有相同的思维方式。和伽利略的差别在于，他把测量距离扩展到了太阳系。在那样巨大的尺度下，他才得以完成光速测量。
罗默长期观察了在十七世纪被伽利略发现的四个木星卫星之一“木卫一”的月食现象。他为什么观测 ”木卫一”?  因为这颗卫星离木星最近，绕木星运行的公转周期短，出现月食的频率比其他木星卫星高，观察就相对容易得多。通过观察他发现，“木卫一”的月食持续时间随着地球和木星之间的相对位置变化而变化，并且这一变化呈现周期循环。当地球在轨道上接近木星时，“木卫一”月食的持续时间比较短，而当地球背离木星时，“木卫一”月食的持续时间则比较长。他把地球接近和背离木星过程时月食的持续时间分别定义为t1和 t2 ， 在这两段时间里地球运行的距离分别为d1和d2， 那么他计算光速的方法可以用以下公式表达:

科学的进步，不仅仅改变了我们对事物本质的认识，也有助于我们对它们的认识更加精确。对光速的认识也是这样。
一位荷兰科学家惠更斯(Christiaan Huygens)在1690年修正了地球轨道，重新计算了光速的值。但他修正的地球轨道也不甚精确，得到的结果几乎和罗默的结果没有重大什么区别。
测量光速另一个具有里程碑意义的成果来自英国天文学家布拉德雷(James Bradley)。1729年，他引入光行差的概念修正光速，获得了精度很高的光速值
301000 km/s
。
何为光行差? 我们可以这样想象，如果一个人在雨中快速行走，即使没有风，他也会感到雨点以某个角度迎面扑来。而他站着不动时，雨点则会垂直落在他的头上。我们知道，地球环绕太阳做椭圆周期运动，从而相对太阳形成这样的夹角。布拉德雷测量了这一夹角，修正了光速值。
他的光行差概念，还间接影响到二十世纪诞生的狭义相对论。荷兰物理学家洛伦兹(Hendrik Antoon Lorentz)就用光行差的概念推导出狭义相对论中坐标转换关系，以期望解释光速在以不同速度运动的坐标系中保持不变的现象。虽然狭义相对论的发明权归功于爱因斯坦，但其中的变换关系至今仍然被人们称之为洛伦兹变换，以纪念他对物理学的卓越贡献。
第一个在地球上用机械设备成功地精确测量光速是法国科学家斐索(Hippolyte Fizeau)在1849年完成的。从原理上看，他的测量方法和伽利略的方法没有本质的差别。只是在具体测量过程中，他的方法无论是计量距离还是时间上的精度都远远高于伽利略的方法。首先，他用一面反光镜取代了助手，减少人为的反映时间带来的误差。除此之外，他还引入一具精密的齿轮来提高记时精度，或者更准确地说，由于这一齿轮的引入，计时在整个测量过程中已经多余，不需要直接对时间做任何测量，从而不会对光速的测量精度带来任何直接误差。
那么斐索的测量方法的关键环节是什么呢? 让我们来看看他的具体测量过程。他把两架望远镜，一架发射望远镜，一架接收望远镜，安置相距遥远的两个位置上。然后他通过强光源发射一束光到一片半透明的镜片上。该半透明的镜片和发射光束以及两架望远镜的夹角分别都是45°。这样，入射的光束被准确地反射到还处于静止状态的齿轮的缝隙上，继而传到发射望远镜内，透过其中的透镜又被传到接收望远镜内。接收望远镜内部的透镜把这束光聚焦到其中的反射镜上，于是这束光又被原路反射回来，传入观察者的眼里。
他的下一步骤是转动齿轮。当齿轮转动比较慢时，反射回来的光速被齿轮的齿挡住，观察者的眼里是漆黑一片。随着转速的提高，光束即可从下一个齿缝穿过，观察者又可以看到反射回来的光束。如果把齿轮转速再提高一倍，观察者继而从再下一个齿缝中看到反射回来的光束。以此类推，三倍，四倍，等等。
如果我们把两架望远镜之间的距离定义为D, 两个齿缝的间距为d, 齿轮的速度为v, 则他的光速c即可以下计算公式得出:
c = 2D * v/d
我们可以看出，他的测量法的高明之处在于不含有任何明确的记量时间过程。其测量精度完全取决于三个参数，而最关键的参数就是齿轮转速，时间记量也隐含在这个过程中。
上面的公式还可以进一步简化为:
c = 4D * n * f
其中n是齿轮的齿数，f是齿轮的转动速度， 单位是赫兹。他的测量数据分别为:
D = 8.633 km,  n = 720,      f =  12.6 hz
所以斐索测得的光速
c = 313 274 km/s
。他的测量原理与伽利略的方法没有本质的差别，但由于测量设备的改进，其精度也大大改善。
两年后，既1851年，斐索拓展了他的测量设备，尝试着测量光在水中的传播速度。为此，他在测量系统中增加了两面反射镜，并在光的行程路上放置了两只水箱。水箱的外壁自然是透明的，以便让光穿行而过。水箱被平行地放置光的行程上，一只在南部，另一只在北部。水在其中流动，但方向相反。水通过一根导管进入南部水箱的西端，流过水箱，然后从东边流出，再然后顺着另一根导管流入北部的水箱的东端，通过水箱后从西边流出。由于两只水箱的几何尺寸一样，从而保证了两只水箱中的水流速一样，但方向相反。
在测量过程中，强光源发射一束光，从西向东射到第一面放置于西南角，面向东北，与东西方向构成45°夹角的半透明的镜子上。光束入射到此镜上，被分成两束。
第一束穿过镜子继续向东，进入南部的水箱，继而入射到一面放置在东南角的完全不透明的反射镜上。这面面向西北的反射镜与东西方向也构成45°，平行于第一面半透明的镜子， 从而保证这束光被准确地反射到正北方向，射入放置在东北角的第三面不透明的反射镜上。由于这第三面镜子的放置角度面向西南，与东西方向构成135°，继而把这束光折射到正西方向，穿过放置在北端的水箱，进而入射到第四面不透明的镜子上。这面反射镜的放置角度面向东南，平行于第一和第二面镜子，于是把光束发射到正南方向，其中一部分光通过第一面半透明的镜子，传到安置这面镜子南端的观察镜内。
而另一束光则被第一面镜子反射到正北方向，先后被西北角，东北角和东南角的反射镜反射，穿越北端和南端的水箱，最后被北置于西南角的半透明镜子反射到观察镜内。
这两束逆向而行的光束最终在观察者的眼中形成干涉条纹。
显然，这是一次不很成功的尝试，所以鲜有文献记载这一实验，也没有留下实验数据可供查询。但它却间接印证了光速不变原理，为1887年的迈克耳孙－莫雷实验(
Albert A. Michelson
，
Edward  W. Morley
)  以及1916年 爱因斯坦(Albert Einstein)创立狭义相对论埋下了伏笔。
也许是直接受到斐索的影响， 另一位法国科学家傅科(Léon Foucault)也对光速测量产生了兴趣。他在1854年改进了斐索的齿轮测量法，进一步提高了光速的测量精度。他的具体作法是用一只可以旋转的反射镜代替旋转齿轮。
在测量过程中，他把一束光发射到那一面可以旋转的反射镜上。这束光继而被反射镜从另一个角度反射到一面曲面反射镜上。该曲面反射镜的曲率半径等于它和那面可旋转的反射镜之间的距离。这样可以保证入射过来的光束被百分之百地反射回到那面旋转反射镜上。如果这面可旋转反射镜处于静止状态，从曲面镜反射回来的光束会被准确地反射到光源上。当这面反射镜高速旋转起来，反射回来的光束就不会回到光源，而是与光源错位一段距离。他准确测量出这一错位距离和那面反射镜的转速，继而计算出光速的值:
c = 8π * f * r * L/∆
在这个公式中， f是旋转反射镜的旋转频率， r和 L分别是光源与旋转反射镜以及旋转反射镜和曲面反射镜之间的距离， ∆则是那一段小小的错位。他给出的测量数据为:
f = 800 hz,  L = 20 m,  r = 1 m,  ∆ = 1.336 mm
所以他测得的公式值为
300900 km/s
。
这里需要提醒一下， 旋转反射镜每秒旋转800转，即使在技术高度发展的今天，这一转速也是一极高的转速。傅科在十九世纪究竟用什么技术方法使反射镜达到这一转速，仍然使我们感到困惑。
二十世纪初叶是科学发展史上发生翻天覆地变化的年代，经典物理大厦在微观和宇宙级尺度下坍塌，量子理论和相对论相继诞生，光速的测量精度随之进一步提高。
从1926年开始，美国科学家迈克耳逊(Albert Abraham Michelson)多次改进了傅科的测光设备。首先，他用一只具有八个反射面棱镜取代了傅科的可旋转的反射镜。当然，这面八面棱镜同样可以旋转，所以当它转动一圈，就相当傅科的反射镜旋转八圈，把入射光反射到另一个角度的可能性提高了八倍，精度也随之提高八倍。其次，他把测量设备中的反射镜的数量增加到七面，让光在各个反射镜之间来回穿梭，其总行程提高到35公里。
于1930年，他做出了他一生中精度最高的光速测量。在他的测量设备中，他抽空了一根长达1.6公里的管道，让光束在反射镜之间来回巡回十次后，最后抵达一只具有三十二面反射面的棱镜。经过长达1900次的测量后，他得出了光速
c = 299774 km/s
的结论， 并且正负误差在
11 km/s
之内。
也正是迈克耳逊在1886年和另一位美国科学家莫雷(
Edward Williams Morley
)合作，应用原理相似的实验设备，证明了当时物理学界普遍猜测的光的传播介质“以太” 并不存在，以及光速在不同运动系统中保持不变之原理，为狭义相对论的诞生奠定了坚实的实验基础。
随着科学的进步，人们对光的本质也有了更深入的了解，渐渐不满足于用机械设备和光学透镜来测量光速，因为即使高精密的机械仪器和透镜也不能完全消除误差。于是科学家开始尝试利用光的物理特性来测量光速。
早在1864年，英国苏格兰物理学家麦克斯韦(James Clerk Maxwell)发表一组优美的关于光的积分方程组，不仅仅精确的描写光在媒介中的传播速度和波动特性，还有机地把光和电磁波统一起来。在随后的二十世纪初叶1905年，爱因斯坦关于光的光电效应的论文，初步揭开了光的量子特性的神秘面纱。1913年，
丹麦
物理学家
玻尔
(Niels Henrik David Bohr)建立的氢原子模型，进一步定量地诠释了光的更深层的量子特性。
于是，光看起来互相矛盾的波动性和量子性有机的结合就展现在人们眼前。
究竟科学家们怎么利用光的物理特性来测量光速呢?
英国物理学家埃森(Louis Essen)率先利用光的波动特性来测量光速。1950年，他和同事戈登-史密斯(A.C. Gordon-Smith)建造了一台圆柱型的空穴共振器，让光在这个共振器来回往返，产生共振效应，在实验室里成功地测量了光速。
在实际生活中，我们经常看到水波在池塘中传播。水波一旦碰到池塘边缘，立即被反弹回来， 与延绵不断入射过来的水波碰撞，叠加。在特定的频率和边界条件下，两束水波的叠加会形成看上去静止不动的波峰和波谷，既驻波现象。

由于用于制作共振器的材料对光的反射有不可忽视的影响，从而对光的共振特性也有影响，相应地降低了测量精度，所以在实际测量中，埃森预先对材料的光反射特性进行了多次测试，并结合理论公式修正了材料对光共振造成的误差，所以他测得的光速值
c = 299752.5 km/s
和现在得精确值几乎没有差别。
我们这里需要指出，埃森在实际测量中用的不是光波，而是电磁波。这时麦克斯韦理论已经广为流传，为世人所知，光波和电磁波在本质上其实是一致的，在真空中的传播速度完全一样。而在实验室里，调节电磁波的频率要比调节光的频率容易得多。
埃森的光速值具有里程碑的意义。
光既然在媒介中以波的形式传播，当然就具备波动的所有特征，其中之一就是波的干涉现象。英国物理学家弗鲁姆(Keith Davy Froome)首先利用光的干涉特性测量出光的速度。
在自然界中，波动的干涉现象就是两束光或者两束波在空间相遇后相互叠加，形成强弱不同的光谱或者波普。特别是，当两束光的从光源发射出来，其频率和振幅完全一样，就会在空间或者光栅上某些位置上相互加强，形成明亮的条纹，而在另外的位置相互减弱，呈现出暗纹。正是这些美观的明暗相间的条纹固化了科学家头脑中光的波动性的观念，以致于他们当中许多人顽固地不愿意承认光的量子性，尽管他们后来面对着许多新观察到的用波动性无法解释的光现象，如电磁辐射能，光电效应，等等。当然， 这是后话，这里按下不表。
弗鲁姆在1950年设计一台干涉仪来测量光速，或者更准确地说，测量频率为72兆赫兹的电磁波速。也许这一台干涉器的测量结果不太令人满意，所以接下来的一年，他重新设计一台更容易操作，精确度更高的干涉仪，并把测量结果发表出来，
c = 299793.0±0.3 km/s
。
让我们来看看他的这台新干涉仪是怎么工作的。当仪器工作时，其中的电磁波发射器发射一束电磁波，经过一道T-型导管后被分成两束波。这两束电磁波首先沿着各自的U-型管相背飞行，
然后通过各自的U-型管出口的发射喇叭，穿过装在接收器上的接收喇叭，产生球面波，继而在接收器中迎头相遇，干涉叠加，相互增强或者相互减弱。
众所周知，磁场在转动的线圈中可以转换成电流，强磁场产生强电流，弱磁场产生弱电流。当他把接收器沿轨道缓缓地移动到两束电磁波互相抵消的位置，电表里电流也缓缓减弱，直至为零。他继续移动接收器，电表就交替显示出强电流和零电流。他记录下第一次和最后一次出现零电流时接收器的在轨道上的位置，以及在这两个位置之间零电流出现的次数，然后用下面的公式计算出光速:

c = 2(Z2-Z1)*  f/N
但在实际测量中，弗鲁姆不仅仅要考虑仪器的变形率，而且还要修正电磁波在仪器中的折射率带来的误差。
从弗鲁姆的推导过程中，我们也可以看出，他使用的电磁波的频率和光在测量仪中空气的折射率是当时已经是可以精确测量的常数，这是他测量光速的理论前提。这也常常使后人惊叹不已，在光速都没有精确计量的条件下，科学家就已经为光和电磁波奠定了坚实的理论基础，麦克斯韦积分方程组更是被称之为科学史上最精美的方程组之一。
弗鲁姆测量的光速值，离最终的精确值只差一步之遥。科学家们究竟怎样迈出了这最后的一步?
1972年，美国科学家埃文森 (K. M. Evenson) 与他的同事们彼得森(F. R. Petersen)，巴杰 (R. L. Barger) 和霍尔 (J. L. Hall) 迈出了这一步，登上了光速测量的巅峰。其实，他们用的公式很简单，既光的频率f乘以光的波长λ:
c = f * λ
但他们使用的测量仪器却是精度极高的激光干涉仪，测出的频率和波长的精度达到十位有效数字，可能是整个科学史上测出的有效位数最长的数值。
激光干涉仪是在波尔原子模型基础上的新型量子光源，与普通光源相比，有着许多奇特的性质，波长，频率单一，偏振方向一致，亮度也很高。埃文森们就是利用了这些性质，精确地测量光速。
激光的理论基础是量子性，而频率和波长则是波动的基本参数。就这样，两个曾经异常尖锐对立并引起众多科学家激烈争论的看起来互相矛盾的观点被和谐地统一在激光干涉仪中。
这是一台氦氖激光干涉仪， 其主要零部件除了干涉仪外，更至关重要的是一台充有甲烷的激光调制器。它不仅仅能够稳定激光束，而且还能调制本来已经很单一的激光频谱，使测量误差范围进一步缩小，大大提高了测量精度。配上一支精密的原子计数表，他们测出的激光频率拥有十一位有效数字的超级精度
f = 88.376181627 THz
， 让人啧啧称奇。同样由于饱和甲烷的稳定性，激光的高亮度以及高度空间相干性，他们测得了人类有史以来精度最高，有十位有效数值的波长
λ = 3.392231376 µm
。 把它与上面的激光频率相乘， 他们特殊测出了人类有史以来精度最高的光速
c = 299 792.4562 km/s
，而且正负误差小于0.0011km/s。
这是科学史上在理论飞跃和技术进步交相辉映下诞生的惊人成就。
这里我们需要指出的是，就在同一年，即1972年， 另一个美国科学家团队(Z. Bay, G. G. Luther, J. A. White)也用激光干涉仪测量了光速。只是精度稍差，所以他们的结果现在几乎淡出了人们的视线。
在随后的岁月里， 虽然世界各国许多实验室验证了埃文森团队的测量结果，但各个数据之间总有一些差别，科学家再也没有测得更精确的光速值。自从爱因斯坦创立狭义相对论以来，长度单位米(m)就和光速紧密地联系在一起。如果每次测得的光速稍微不一样，就意味着长度单位米(m)有所变化。同时，当科学家假定光速为固定值，测得的波长比在原有的长度条件下可重复性要高得多。于是， 在1983年的第十七届国际计量大会上，世界各国的科学家达成一致意见，把光速规定为
c = 299792.458 km/s
，而一米(m)则是光在
1/299792458 s
时间内经过的距离，从而成为依赖于光速的一个计量单位。
从此，光速就永恒地固定在
c = 299 792.458 km/s
。
