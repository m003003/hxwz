1

道理讲得已经够多了：过去的已经过去，世上没有后悔药可吃；未来的尚未来到，忧心忡忡也是徒然。美国人说：The past is history, the future is mystery, only the present is a gift。人唯一能做的只是珍惜当下的拥有。《日瓦戈医生》的作者Boris Pasternak 说过一句话：“人来到世上是为了活着，不是为了准备活着(Man is born to live and not to prepare to live) 。”

可是说起来容易做起来难，因为眼睛看得见的当下都是些不重要的东西。抬眼看看我当下的周围：沙发、客厅、地上的纸张、外面或晴或阴或雨的天，这些都不可能比我脑子里此起彼伏汹涌澎湃的各种念头在我的这个当下之中占据更重要的地位。当下的99%的内容是自己。所谓当下所拥有的，不管是钞票、蓝天绿草还是什么别的福气，都是故事里的配角。主角是那个大大的 “I”。所谓的境随心转就是这个意思。不管这个境有多么浩瀚，只要我的心是晴的，境就是晴的，只要我的心在泣血，境就在泣血。

生活在近五百年前的蒙田 (Montaigne) 可能是观察自己的先驱。他说：别人都往前面看，我是往里面看。如果一个人只知道往前面看，他就没有看到屋子里的大象，他的当下就是个不值得一提的干瘪小角落。什么时候能看得见了被心事纠缠得青筋暴跳的自己，看见了这图景之中的幽默，胸中的汹涌澎湃就慢慢平静下来了。于是当下拥有的蓝天绿草和各种零碎东西就又都变得楚楚可爱了。不懂得往里面看，就不会懂得珍惜当下。

以前刚听到冥思 (meditation) 这个说法的时候，觉得好像是类似于发功运气之类的玄虚东西，后来读印度思想家克里希那穆提 (Krishnamurti) 的《人生中不可不想的事》，对冥思的意思略有所悟。其实冥思很简单，就是静静地观察自己的各种念头、懊恼、妒火、欲望、怒气、忧心忡忡。这些都是实实在在的、天天忠实地跟着自己在人生战场上拼杀的将士，跟经脉气场之类的玄虚之事没有任何关系。冥思不需要一个绝对安静的角落和一大段闲人莫扰的时间，也不需要努力排空所有的杂念 C 我想，人脑子里冒出的所有念头，不管是高尚的还是卑劣的，都是平等的，各有各的深刻渊源，不好把其中的某一些扣上“杂念”的帽子。况且，要想驱除“杂念”就要用强力，这是一件不自然的事，结果只能使人更躁动。并且这强力本身就是一个大大的杂念。

前几天正在读蒙田的“论虚荣”，猛然意识到自己以前做过的很多事，不管当时被冠以如何美妙的动机，其实都是被虚荣心驱使的结果。这发现让我的那一天大大释然。虚荣心的公式里的关键一项是自己跟别人的比较。看到哪些人比自己强了，就有点不是滋味，于是想要做些什么事来证明自己的力量。当下的状态是自己不如别人，于是就试图做些什么事来扭转这个状态，这就是不接受当下。我明明相信每个人都有自己独特的价值，却又每每在下意识里把自己跟别人比较，这是从小留下的后遗症。我想每个人或多或少都有各种各样从小留下的后遗症，要靠自己有意识的努力来治愈。当然，努力的前提是自己要知道自己有后遗症，所以只知道往前面看的人恐怕就要扛着后遗症一辈子了。

观察自己的一大好处是把自己请下神坛。人站在高高的神坛上被自己膜拜其实并不那么自然，站低一点要舒服得多。

2

发现了自己从前总被虚荣心驱使，以后应该就会少一点被虚荣心驱使，我也就从此又多了一点自由。这是自己给自己的自由。自己给自己的自由是最大的自由，别人给的自由都是假的。记得刚来美国留学的时候，对什么事情都要自己来张罗很不习惯，便联想起国内电视剧上讲的美国社会的冷漠。后来在美国呆久了，才知道中国是个比美国冷漠得多的社会。我初来美国时的不习惯只是因为习惯了在国内时每个人都被制定了一条固定轨道的做法。绑在轨道上的人被释放出来，这自由是别人给的，所以反而不习惯。

中国社会的冷漠已经初具国际影响了。一位白人朋友去纽约的中国领事馆办签证，回来跟我说：美国土地上唯一一个有公开种族歧视的地方就是中国领事馆，并且这歧视不是对他这样土生土长的美国公民，而是对领事官员们的自己人――中国人，或者说大陆人。我自己也去过中国领事馆，观察是也不能全怪领事官员，因为确有些“自己人”顾客置规则条文于不顾，死缠滥打，搞得领事官员不得不板起了脸硬起了心，不给自己人半点通融的余地。见了外国人笑容可掬，见了自己人就恶向胆边生，这是可悲的现实。

现在很多人都在疾呼中国是个不自由的国家，但我猜想，哪一天中国有了言论、新闻、结社和游行的自由，很多人可能却要大大地不习惯一阵子。在独裁社会中，人的恶性是被峻法压制的。在自由社会中，人的恶性多半要靠自己的觉悟来克制，靠法律约束的部分只是一少半。不自由的人被别人输入了自由，他的各种恶性就会骤然爆发出来。民国带给了中国一点自由，其结果是让最藐视自由的一帮坐了金銮殿。这不等于中国人不配拥有自由，只是被别人赐予的自由是假的，自己体验过和习惯了的自由才是真的。韩寒说：“当街上的人开车交会时都能自觉关掉远光灯的那一天，中国人才可谈革命论民主”，说的就是这个意思。这一天可能很遥远 C 其前提是每个中国人都能自己给自己一点自由。如果一个有志于改革社会的人相信只要换掉城头大旗就能给老百姓以自由，那么他的那一套恐怕就与土匪无异，也不可能带给他的同胞任何福分。

人做事有两种动机：一种是出于对自己生存状态的不满，一种是出于爱。虚荣心是怕别人比自己强，恐惧感是怕将来丢掉现在拥有的东西，贪婪是觉得手里玩的人和物还不够多不够好，这些都是对自己当下生存状态的不满。世界上所有的战乱、迫害和不平，包括中国的每一次改朝换代（辛亥革命除外）都是被这种动机驱使。如果人接受了自己当下的生存状态，懂得了活在当下，他就是个满足的人，他的做事动机就是出于爱：对他的事业的爱，对他的同类所遭受苦难的同情。出于爱而工作，每个时刻都是享受，所以每天工作十六个小时都不会觉得累。一个社会的真正前进靠的是后一种动机。中国也有这样的人，只是太少，并且经常充当了悲剧角色。

活在当下、观察自己、往“里面”看、冥思说的都是一个意思。这可以说是一种心灵的锻炼。锻炼心灵与锻炼脑子不一样：锻炼脑子的活动如数独和魔方之类是往前面看的活动。脑子好使的人可以看到宇宙的尽头，但还是可能一点都看不见自己。想身体强健就要锻炼身体，怕老年痴呆就要常动脑子，要想有强健的心灵就要练习活在当下了。心灵强健了，自己给自己的自由就越多，就有可能在任何冷漠的沙漠之中都求得平安。他们还能心甘情愿地为这沙漠增加一点绿色而不求给自己的回报。刘晓波就是这样的人。懂得锻炼身体的人最多，懂得锻炼脑子的人较少，懂得锻炼心灵的人最少。

不懂得享受当下的受害者之一是脸。人要是总不满足，日久天长那一份压力就会刻写在脸上。有一次停车在纽约唐人街闹市，打量眼前掠过的无数张脸孔。其主色调，套用一句早年的流行词，就是苦大仇深。从后视镜中照照自己，我也不是个例外，尽管那时已经开始懂得一点享受当下的道理。

心灵强健了，脸就开始起变化。把德雷莎修女放在一群同样装束的老妇之中，我想很多从没见过她的照片的人也能一眼就认出她来：心灵强健的人那种清澈的眼神是装不出来的。

3

最懂得活在当下的是孩子。一位同事跟我说：孩子是父母的老师，因为他们根本不关心父母的烦恼！我就是从我的孩子那张天真平安的脸上开始懂得活在当下的道理的。我从小学就开始失眠，前后二十年时间，到有了孩子后嘎然而止。

我想我对我的童年有一种仇视。小学就开始失眠 C 这样的童年显然不是什么值得眷恋回忆的日子。现在我自己的孩子们正是在我已经失眠的年纪。每天晚上，我通常是最后一个上床睡觉，路过他们的卧室时能听到他们此起彼伏的微鼾，那在我听起来就像是最美的音乐。这音乐让我骄傲。

虚荣心是个代代传承的文化。我的虚荣心大概来自于虚荣心曾经极重的母亲。母亲在退休之后才对她的虚荣心有所反省，那自然对她自己是个大大的解脱，但那时我早已独立生活了。大而言之，中国人是虚荣心极重的民族。搞奥运、搞全运、搞世博、搞登月、追捧国际奢侈名牌，都是出于虚荣心，要让别人看到富起来了的王老五的架势。至于农村的穷人、污浊的空气、黄绿色的河流，那是我管不着的闲事。有虚荣心的人最怕别人说他虚荣，所以只有一个机缘能把他从这重负之下解脱出来：他能看到自己的里面的时候。

孩子能活在当下，只是因为他们不懂得过去和未来，所以那只是个基本的生理本能。成年人已经知道了过去和未来，想要退回到孩子的认知水平是不可能了，所以只能学更高层次的活在当下：我还记得过去的遗憾，也牵挂着将来的变数，但我还是有能力享受当下。这就像是先从薄读到厚，又从厚读到薄的一本书，是跟孩子那一册本来就薄的书不一样的。这样的活在当下就是比基本生理本能更经得起摔打的活在当下。

现在自己捡回来了一点赤子之心，于是我便努力让我的孩子们的生理本能变成经得起摔打的活在当下的能力。从三件事上做起：

节制虚荣心：抑制把自己孩子与别的孩子比较的冲动。不用说某某孩子如何好的办法来激励孩子，也不夸赞自己孩子比别的孩子强。

节制贪欲：不给孩子买太多的时尚产品，鼓励他们在简单的生活中找到快乐。我的孩子们的“可炫耀品”是他们的同伴中最少的，尽管他们为此时有牢骚。

驱除恐惧感：不用激将法来迫使孩子做他们不敢做的事，但是鼓励他们面对自己的恐惧感。如果能成功克服恐惧心理而做成一点事，可以得到奖励。

不过，要是父母想让孩子读点书，而孩子只想玩电子游戏，怎么办？当民主国家的领袖比当独裁者要难得多，当个希望尊重孩子权利的父母也比当虎妈狼爸要难得多。我的办法是与孩子讨价还价。世界上每个人都想法各异，所以世界上永远都会有冲突。非暴力解决冲突的最好办法是讨价还价：双方平等、绝不强迫、各摆理由、各有让步、好说好散。父母与孩子的区别只是知道的多与少，没有地位上的不平等。既然地位平等，就不能强迫。既然父母自恃懂得更多，就该有以理服人的信心。既然父母想让孩子让步，自己就也该懂得让步。

父母为孩子付出很多 C 父母有权让孩子知道这一点，所以也有要求孩子听一些话的权利。反过来，如果父母能意识到孩子也有还价的权利，就会发现孩子其实是通情达理的。孩子通常都比父母更通情达理，因为父母通常都有以势压人的潜意识。成年人总在抱怨自己得到的自由太少，却想不到他们恐怕并没有给予别人 C 包括他们自己的孩子 C 足够的自由。懂得了享受当下、懂得了看到自己的里面，自己才能多一点自由，才能给孩子一点自由。他们周围冷漠的沙漠也会开始多了一点绿色。
