1

比起现在国内大城市孩子们的学业负担来，我小时候可以算得上是身轻如燕了。那时有无穷无尽的课余时间，经常要靠呆望天空来打发。父母整日奔波于工作、家务和政治学习之间，而且并没有什么望子成龙的雄心壮志，所以是没有工夫也没有动力管孩子。我爱读书，可是能找到的书很少，只能饥不择食，绕着坊间的《西游记》、《水浒》和《三国》这些书打转。除了半白话的原版，《三国》还有一套在孩子中间极为流行的连环画册，《西游记》还有电台播出的改编长篇快板书，当然还有给我留下深刻印象的动画片《大闹天宫》。

《大闹天宫》的天宫美景让我心驰神往，但有些情节让我不舒服。我不能理解那位主人公的做事原则。孙悟空想搞一件兵器，仗着拳头硬，去龙宫要人家把看家宝贝一件件拿出来试，还挑肥拣瘦。拿到了兵器还要要行头，不给还要说人小气。明摆着是孙悟空仗势欺人蛮不讲理，跟我们这里街上的流氓有什么两样，怎么就成了英雄。这不是让我心仪的那一类英雄。那几个龙王只是想保护自己的财产，并没有做错什么，倒成了坏人和丑角。悟空到了天上，听说弼马温原来是个小官、不够接到生日晚会请柬的级别，就去把人家的晚会给砸了。那时的通行说法是：龙王和玉皇大帝代表的是封建剥削阶级的统治，所以悟空去砸那叫替天行道。我的疑问是，被压迫阶级就该靠这样的耍无赖来行道吗。这山大王懂得哪门子的天和道。他打砸抢的目的很单纯：给自己捞家当（金箍棒）、嫌社会地位低（弼马温）或是被扫了面子（没接到生日请柬）。

《水浒》中让我觉得不舒服的人物是李逵。且看这位英雄出场的形象：

“李逵道：‘没奈何，且借我一借，明日便送来还你。’小张乙道：‘说甚么闲话?自古赌钱场上无父子，你明明地输了，如何倒来革争？’李逵把布衫拽起在前面，口里喝道：‘你们还我也不还？’小张乙道：‘李大哥，你闲常最赌的直，今日如何恁么没出豁？’李逵也不答应他，便就地下掳了银子，又抢了别人赌的十来两银子，都搂在布衫兜里，睁起双眼，就道：‘老爷闲常赌直，今日权且不直一遍。’小张乙急待向前夺时，被李逵一指一交。十二三个赌博的一齐上，要夺那银子，被李逵指东打西，指南打北。李逵把这伙人打得没地躲处，便出到门前，把门的问道：‘大郎那里去？’被李逵提在一边，一脚踢开了门，便走。那伙人随后赶将出来，都只在门前叫道：‘李大哥，你恁地没道理，都抢了我们众人的银子去！’只在门前叫喊，没一个敢近前来讨。”

赌博是出于各人自愿：赢者愿打，输者愿挨，所以姑且算是英雄的白璧微瑕。但赖帐、抢钱、打人是实在和英雄二字联系不到一起。我读《水浒》那几年正是李逵被党的宣传机器捧为梁山上最大的英雄的时候。我是怎么都看不出这样的人如何就配英雄二字，能替天行出什么样的道。

那时候我的脑子里经常冒出各种各样稀奇古怪的想法，但没有跟任何人交流这些想法的习惯。很久之后我才意识到这有多么不正常 C 根据我对下一代的观察，孩子的天性是不隐藏什么的。孩子开始对别人隐藏他们的想法，是因为他们的想法被冷落、被嘲笑，让他们怀疑他们自己的脑子是否出了问题。但这些想法不会自动消失 C 我到现在还记得几十年前我对李逵和孙悟空的疑问，尽管那时这两个念头可能只是短短的一闪。人的这些不会自动消失的想法就像河里的水：表达出来、交流出来，河水就顺畅地流走；没有表达和交流的渠道，河水就没有了去路，水位就会越来越高。到了水坝不胜其负的那一天，自己和社会都要为此付出代价。人常说青春期孩子的父母一定要做好的听者，我说等到孩子到了青春期父母才想起来做听者就晚了。那时孩子就已经牢牢养成了在很多大事上不跟父母交流的习惯。在我怀疑孙悟空和李逵的英雄资格问题的时候，我的父母忙着他们的工作、家务和政治学习，我的同辈人忙着膜拜模仿各种各样有武力的英雄，显然没有人会理睬我这没来由的疑问。我只能断定我自己不正常。于是这些疑问就悄悄蓄在我那水坝的后面。

小时候看人通常只是看到两极：一个人要么就是英雄，要么就是坏人。孙悟空和李逵被宣传为英雄，我就带着英雄的期望值去看他们。从这个角度上说，收获失望不奇怪。吴承恩和施耐庵没有把孙悟空和李逵写成高大全的英雄，这一点上他们比样板戏的创作者和欣赏者要高明不少。不过两位作家显然是把无赖当有趣来写，他们不觉得英雄把小民揍得屁滚尿流有什么不正常。由此也可知屁民不是本朝才有的事 C 屁民就是被有权或有拳者揍得屁滚尿流的小民。

有人可能会反驳说：龙王和玉皇大帝怎么是小民。不错，龙王和玉帝都是有头脸有地位的人物。不过，比起孙悟空的本事来，这二位不都是做陪衬的丑角吗。所以孙悟空眼里的龙王和玉皇大帝跟李逵眼里的张乙之辈或得势农会眼里的地主富农没什么分别，都是些一脚一溜跟头的屁民。吴承恩笔下的世界就是个拳头即真理的黑社会，悟空的一身武艺最后还得要手段更高的西天如来出手才被搞定在五指山下。初生“猴犊”最终被同化，不是因为被那里的真善美感化，而是被老大的紧箍咒驯化了。

施耐庵之后又过了几百年，到儿时的我这里，我周围的人还是不觉得英雄把小民揍得屁滚尿流有什么不正常。他们从小到大都生活在不公平的社会里 C 这是他们见过的唯一的一种行事方式。流氓习气被认为是一种纯粹正常的社会行为，因为大家不知道什么是文明。况且，既然这社会从来都不公平，从来都是像当今的天空那样污浊，那么也没有必要用公平的办法去对付。秩序总是令人憎恨的、对自己不公平的，所以挑战秩序的就是英雄，不管他用的是什么样的手段，也不管他能建立的是什么秩序。抗争的结果也不必公平，只要自己能抢到比别人都大的一份就好了。

我小时候对《三国》的印象要比《水浒》好，可能是因为三国人物没有梁山英雄那么痞气。那时我不理解《三国》里的一个地方是蜀国那绝顶聪明的诸葛亮怎么就屡战屡败，把自己的国家搞得一天不如一天。

2

作家笔下的英雄是作家本人理想的寄托之处。了解了这些人物，也就了解了作家本人。前几年读到托尔斯泰的《复活》，便感到托翁与吴承恩施耐庵虽然同是作家，其差异比他们之间的时空距离不知还要遥远多少倍。书中讲的是贵族聂赫留朵夫公爵担任陪审员时，巧遇当年被自己始乱终弃后又被陷入狱的女仆玛丝洛娃。这巧遇唤醒了他选择遗忘十几年的记忆，开始了他追悔、拷问自己、最后灵魂“复活”的历程。跟三国水浒不同，书中所有的搏斗都是发生在“里面”的。下面是书中诸多“搏斗”场面之一：

“同米西结婚，昨天他还觉得很称心，今天却觉得根本不可能。昨天他认为就自己的地位来说，她同他结婚无疑将得到幸福，今天他却觉得他不仅不配同她结婚，简直不配同她亲近。‘只要她知道我是个怎样的人，就决不会同我来往了。我却还要埋怨她向那位先生卖弄风情呢。不行，就算她现在嫁给我，而我知道那个女人关在本地监狱里，明后天就要同大批犯人流放出去服苦役，难道我能幸福吗？不仅不能幸福，而且内心也不能平静。那个被我糟蹋的女人去服苦役，我却在这里接受人家的祝贺，还要带着年轻的妻子出去拜客。或者，我瞒住首席贵族，同他的妻子无耻地勾搭，同时又同他一起出席会议，统计票数，看有多少人赞成、多少人反对由地方自治会监督学校和类似的提案，事后又约她幽会，这是多么卑鄙呀！或者，我将继续去画画，虽然明知那幅画永远也画不成，因为我根本就不该去干那种无聊的事。事实上我也根本无法做那种事，’他自言自语，由于内心发生的变化而暗自高兴。”（第一部第33章）

发生这搏斗的地方是四才子书很少触及的角落。这角落姑且可以叫做灵魂，不过灵魂在中文里经常指的是人死之后才突然拥有的某种神秘的飘浮在半空中的东西。活人的灵魂不经常被提起，拷问自己灵魂这样的事也不像是脑筋正常的中国人会做的事。托翁笔下的聂赫留朵夫一直在拷问自己的灵魂，而在几本才子书的成百上千个人物里，就算是严于律己者如诸葛亮，也不过是懊悔自己的战术失当。我不记得哪个人物有过认为自己卑鄙的时候。一个社会中没有人认为自己卑鄙，听起来好像是至善至美的大同社会了，但是不认为自己卑鄙和不卑鄙不是一回事。人都有卑鄙的一面。不认为自己卑鄙，只是因为看不到自己的灵魂罢了。

老舍早就注意到（《灵的文学与佛教》，一九四一年）：“从中世纪一直到今日，西洋文学却离不开灵的生活，这灵的文学就成了欧洲文艺强有力的传统，反观中国的文学，专谈人与人的关系，没有一部和《神曲》类似的作品，纵或有一二部涉及灵的生活，但也不深刻。我不晓得，中国的作家为什么忽略了这个，怎样不把灵的生活表现出来？最奇怪的，中国文学作品里没有一部写劝善改恶的东西，很多的书本里，虽也有些写到‘善有善报，恶有恶报’的字眼，但都不是以灵的生活做骨干底灵的文字，至于象阴骘文这类的著作，虽也可称为导人向善的文字，然总不是文学的作品，只不过是一些劝世文罢了。尤令人莫名其妙的，就是那类最坏的书――使人看了会上当的书，也在说是劝善的作品，没有灵的文学出现，怎能令人走上正轨，做个好好的国民？然而就我研究文学的经验看来，中国确实找不出一部有‘灵魂’的伟大杰作，诚属一大缺憾！”

我一直觉得“善有善报，恶有恶报”这样的宣传在中国文明史上的作用很可疑。善报是诱惑，恶报是恐吓。出于诱惑或恐惧去做事是一种投资或投机。将来的那个“报”就是他做事的全部动机。做善事是给自己和子孙积阴德，吃了点亏就大呼好人不得好报。英雄做事不需要这样的诱惑或恐吓，因为他的全部行事标准都出自他的灵魂之中。他做事只因为他认为应该去做，他做事的回报也就在当下：他由于为善而得到灵魂的平安。努力的结果是什么，或将来能得到什么利益都不重要。他的付出是无条件的：他不认为受益于他的人欠他什么人情，也不认为上帝或佛祖或苍天欠他一个善报。他立志去改造社会，因为这让他快乐。这对他就足够了。他也有卑鄙的时候，但他能看得到。他会被他自己的灵魂责备。

四才子书中我最喜欢的人物是贾宝玉。他没有李逵张飞的勇力，也没有诸葛亮的智谋，但有一颗善良的心和以一己之力反叛大流的勇气。遗憾的是他不懂得强健自己的灵魂，于是他只能成为以卵击石的悲情人物。

3

吴承恩与施耐庵是五六百年前的古人，或许不好与一百多年前的托尔斯泰比较。不过，《神曲》的作者但丁便与施耐庵是同时代的人。另外，三国水浒西游在中国的影响长盛不衰，近几十年更被无数次改编为电影、电视剧、各种儿童读物，这说明施耐庵和吴承恩对英雄的理解是被后代的中国人不走样地继承下来了。海外中文学校里一套流行的教材中编入了一些中国上古神话传说故事，诸如水神与火神打架、后羿射日、嫦娥误食灵药等。我教到这些故事时，心想，后羿射日与我做一趟广播体操有什么本质的不同呢。他以他的能力做了一件事，我也以我的能力做了一件事。这样的故事是否有必要占据小学课本页面。教材编撰者显然是继承了施耐庵和吴承恩的英雄观 C 后羿能射日，所以是英雄，值得写到课本里。我不认为后羿是英雄，因为我没有看到后羿的灵魂，或者说，我没有看到后羿有什么让我自愧不如的地方。要是让我去给海外华人子弟编中文教材，我会选“子路、曾皙、冉有、公西华侍坐”这样的故事。我觉得那才是有灵魂的故事。孔子是懂得灵魂的人，可是疯狂烧钱在海外扩张“孔子学院”的一干人是不懂得灵魂的人。孔子学院里边有的是脸谱、风筝、剪纸等中国各种各式民俗花样，偏偏没有孔子的思想。

施耐庵和吴承恩的英雄观自然也不是他们所独创。从他们那里再上溯两千年，看看卧薪尝胆的英雄越王勾践。勾英雄报复仇人的一条著名妙计是把大量煮熟的稻米当种子献给吴国，让吴国来年大片土地颗粒无收。当时吴国是海内大国，那么被这一条妙计活活饿死的老百姓恐怕不是个小数字吧。为报私仇竟然忍心饿死那么多平民，几千年来不但没有人质疑勾践的英雄资格，其故事反倒成了激励华夏后人发奋图强的经典篇章。当然，勾践是古人，那个时候的英雄杀人如砍瓜切菜或许是正常的事。但一直到今天，这样的人一直被后人奉为英雄。中国人的英雄观几千年来并没有什么变化。

没有灵魂的英雄不过是一件出色的工具。唐太宗有豪言：“天下英雄，尽入吾彀中矣！”这比刘邦的“某业所就，孰与仲多”更高明：刘邦把天下当成了他的自家后院，李世民则是把天下英雄的灵魂都圈进了他的彀中。控制别人的财产只是治标，控制别人的思想才是治本。一个人的彀有多大呢？假如但丁先生挟着讥讽教皇的胆量来到大唐朝廷，上书太宗曰：你为争夺权位杀死两位兄弟，悖逆天理，死后当下地狱中最底下一层的该隐狱，永世被冻在暗无天日的冰河之中，不知太宗皇帝对这样的人是否还会有对他彀中别的英雄那样的雅量。从这里不难理解为何唐朝人的思想高度比起春秋战国来判若云泥 C 都到了唐太宗的彀中了。不管他如何开明，一个人的彀又如何能容纳下普天下的英雄呢，愿意入这小小的彀的人是否又能称得上英雄二字呢。

4

小时候看《西游记》还有另一个疑惑：这经是什么宝物，值得费那么大的周折跑那么老远去取。我现在的理解是：宗教是帮助人们探知自己灵魂的努力。聂赫留朵夫的灵魂就是靠《圣经》才得以“复活”，佛经则在中国古代起到了多少类似于《圣经》的作用。基督教的《圣经》只有一本，佛教经典却是卷帙浩繁。这么多佛经能从印度传到中国，玄奘 C 唐僧的原型 C 居功至伟。

玄奘自幼聪慧勤奋过人。唐贞观初年，年仅二十几岁的玄奘已经是名满长安的高僧。苦于当时佛经原始资料有限，矛盾之处甚多，玄奘决定抛下地位崇高的京城生活，直接去印度研究佛经。在与同道联名向太宗请愿西行被拒之后，他独自带几个随从悄悄从长安出发，一路向西绕过整个青藏高原和帕米尔高原，再折向东南去往印度。玄奘此行历时数年，途中穿越沙漠雪山，也并没有武功盖世的孙悟空保驾护航。性命屡次悬于一线之时，灵魂之中该发生了不少搏斗吧。不过吴承恩显然对灵魂之中的搏斗不感兴趣。于是玄奘靠匹夫之力跋涉上万公里为华夏子孙取回经卷的壮举就成了乏味的步行锻炼，不加一点辣椒酱油进去就不成其为故事。结果是虚构的猴子成了主角，中国历史上屈指可数的伟大英雄之一倒成了丑角。中国不是没有英雄，而是他们的同胞不认识他们。

普通人无论如何当不了上天入地的孙悟空，但每个人都有潜能做只身穿越欧亚大陆为同胞灵魂谋来福祉的玄奘。屁民与英雄之间并没有出身的差别。这就是我喜欢托尔斯泰的原因。反过来说，虽然每个人都有潜能，但只有英雄才把潜能变成了现实，完成了马斯洛所谓的自我实现。所以，与其说英雄是人群之中的伟大者，不如说英雄是人群之中的幸运者。迪斯雷利说：“多数人直到进坟墓之时也没来得及唱出他们心中的歌。”借用迪斯雷利的说法，每个人心中都有一首歌，英雄只是有机会把它唱出来的少数幸运儿而已。
