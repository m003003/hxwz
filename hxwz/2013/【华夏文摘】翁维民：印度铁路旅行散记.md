“生活在鱼缸里的鱼可能并不知道鱼缸是什么样子的， 一个旁观者可能知道整体感， 却没有生活在其中的真实感受。” 《大趋势》 的作者： 约翰. 奈斯比特

“印度人是世界上最聪明的民族， 50年以后我们将统治世界。”  一个30多岁，长着大圆脸的医药公司销售员认真地告诉我。当时我们正坐在从孟买(Mumbai) 开往斋普尔(Jaipur) 的二等空调卧铺车厢里。

“那么依你看，到那时候，中国又会处在怎样的一个位置呢? ” 我问到。

“中国会和我们一起领导这个世界。不同的是我们更多地在办公室里用脑力劳动，而中国人则是更多地在工厂的车间里干活。你没听说过吗？ 印度是世界的办公室， 中国是世界的工厂，50年后一定还是那样的。” 一个40岁左右的铁路雇员信心满满地插嘴说。

1853年4月16日， 在21响礼炮声中， 一列拖挂着14节车厢， 载有400名乘客的火车拉响汽笛， 轰隆轰隆地驶向34公里外的塔纳。印度铁路运输由此启动， 它比中国的第一条铁路吴凇铁路 (1876年) 整整早了23年。

到今天印度铁路的总里程已达6.4万公里， 拥有6853个车站， 日发车量达1.5万列， 日运送旅客1千3百多万， 货物130万吨。据印度铁路部门的官员估计， 至少有8千万人靠铁路为生。英国人早就离开了，但他们给印度留下的议会制度，英语和铁路已经在印度深深地扎下了根，成了几乎每一个印度人引以为傲的国家标志。美国地理杂志更将其称为世界上最伟大的铁路系统。

印度从来就是一个等级森严的国家， 曾以四大种姓制度闻名于世。今天虽然在法律上已是人人平等， 但只要一看姓氏，你的祖先所属的种姓就一目了然， 无所逃遁了。谈话对手的语音和声调，往往在知道你的姓氏之后会有难以察觉的微妙变化。印度的铁路系统自然也不例外， 有着令人眼花缭乱的车厢等级。

Super Fast 特快， Fast 直快， Express 快速， Passenger 普通客车， Metro市区列车。当你化了九牛二虎之力，刚从那密密麻麻的火车时刻表上找到你要的那列车之后， 你还得伤脑筋，来决定要哪一个等级的车厢。印度的列车车共分8个等级:  1st A/C tier 一等空调卧铺，2nd A/C tier 二等空调卧铺， 3rd A/C tier 三等空调卧铺， Sleep Class 硬卧， AC Executive Chair Class 空调行政软座， AC Chair Class 空调软座， 1st Class 无空调一等硬座， Unreserved 2nd Class 无预定二等硬座。相差一个等级， 票价就会相差几乎一倍。

印度的长途列车上只有烧饭的餐车， 没有吃饭的餐车。通过预定，各种餐点会很快地送到你的面前。一个长方型的铝皮盆子， 里面大大小小的塑料盒子装着米饭， 烙饼， 辣酱， 咖哩鸡， 炖烂的豆子， 杂绘蔬菜糊， 还有塑料薄膜包装的150毫升饮用水， 这就是一份典型的印度列车正餐。

一手拿着泡沫塑料杯子， 另一手拿着暖瓶的小伙子，则是几乎每个小时都要来回走上一遍， 口中不停地吆喝着：“茶! 茶! ”  茶的栽培和制作技术，是当年英国人从中国偷去， 再移植到印度去的，匆忙之中，甚至没来得及给茶取个当地语言的名字， 这么多年来仍和中国人一样， 茶啊茶地叫着， 真是令人难以相信。

“Hello, 你去哪里?” 一个穿西服的中年男子问道， 将我从对车窗外的凝视中唤了回来。

“ 阿格勒 (Agra)， 去看泰姬陵(Taj Mahal)。你去过泰姬陵吗? ”我回答说。

“没有， 你知道吗? 泰姬陵仅仅只是一座坟墓！我是不会专程去看一座坟墓的。” 那个男子用带着劝戒的口气对我说。

半个多世纪之前， 鲁迅怀着对国人种种性格弱点的沉痛心情，写下了(( 阿Q正传 ))， 可惜的是，鲁迅当时没有机会到印度来走一回。在这儿几乎每个人都可以做阿Q 的师傅。我看这个世界上没有任何其他的民族像印度人的自我感觉那么好，看着他们那永远信心百倍的脸， 听着他们那些不着边际的夸夸其谈。真的你不禁地要问自己， 生命是那样的短促， 为什么我们不能像印度人那样快快乐乐地过一生?

泰姬陵被火车上的那个男子轻描淡写地称作只是一座坟墓，是的，它是一座坟墓。 但他不清楚这可是世界上最伟大的坟墓之一。和埃及的大金字塔一样， 它们是人类智慧的最高结晶。如果说大金字塔粗圹雄浑， 气贯长虹，像一个巨人般地屹立在尼罗河畔。泰姬陵则风格端庄，秀丽，充满灵秀之气，宛如一位绝代佳人。以白色大理石为主构筑的泰姬陵，以其女性的柔美和清雅，给人以水样般的纯洁和宁静， 同时又充满了皇家典雅恢弘的气度。

关于泰姬陵还有一个美丽的传说。泰姬陵始建于1632年，正值印度历史上莫卧儿王朝的鼎盛时期，当时的国王沙•贾汗是一位励精图治，很有作为的君主。他有一个叫姬曼•芭奴的宠妃，不仅貌如天仙，而且聪明贤惠。姬曼•芭奴入宫后，深得沙•贾汗的宠爱，先后为国王生了14个孩子。沙•贾汗赐给这个宠妃一个非常美丽的称号：“泰姬•玛哈尔”，意思是 “宫廷的王冠”。国王与爱妃情深意笃，形影不离，就是外出巡视时也要把她带上。1631年，泰姬•玛哈尔随沙•贾汗出巡时，因中途生子难产，竟香消玉殒，终年才38岁。在她弥留病榻之际，国王问她有什么要求，她说：“陛下若不忘我，请为我造一座大陵墓，以此来纪念我们真挚的爱情。” 悲痛欲绝的国王遵照她的遗言，为她建造了一座像她一样美丽、举世无双的巨大陵墓，并用她的封号命名，简称泰姬陵。

在印度坐火车旅行，往往会得益于它那效率极高的电脑定票系统， 你可以在任何车站订购90天之内任何车次和等级的车票，你不仅仅可以在列车的始发站买到对号入座的票，就是中途上车也没有问题， 同样都是对号入座。当我准备从桑塔(Santa)乘车到圣城瓦拉纳西 (Varanasi)时， 售票员告诉我，第二天的所有等级的卧铺票均已售完了， 但他可以把我放在等候名单上。我问他如果放在等候名单上的话， 有多大的可能我可以拿到自己的铺位，回答是95%。果然第二天上车前，我的名字和相关的卧铺号码已清清楚楚地出现在车站的告示牌上了。不知道这整个系统是如何运作的， 但其效果确实令人印象深刻。

虽然在所有的大站都有专供外国人买票的窗口，手持外国护照就 可以不用排长队，但进站就得走进印度人的洪流。 无论站内站外， 白天黑夜， 永远是万头攒动， 人声鼎沸， 仿佛全印度的人都在坐火车旅行。令人感到意外的是，虽然人流如潮，可秩序井然。无论进站还是出站，都没有人检票。就是上了车， 查票的工作人员往往过了1-2个小时才来。我曾问同车的乘客， 难道就真的没有人无票乘车， 或者以低等级的车票去坐高等级的车。他们听到这样的问题，无不露出惊讶的表情， 怎么可能呢？ 什么样的票就坐什么样的车， 这就象太阳总是从东方升起， 那是完全不容置疑的，没有人不感到奇怪，我竟会提出这样无聊的问题。在那瞬间， 我在周围印度人眼里大概就象外星人一般的可笑和无知。

在首都新德里的火车站台上， 站着一位身高1.80米， 体型魁梧，满脸浓密的黑色胡须， 裹着青灰色的锡克族头巾， 穿黄色军服的年轻陆军军官， 正耐心地一次又一次地用脚将一只围着他的行李箱打转， 肥胖得行动迟缓的老鼠拨到一边去。坐在我右边， 一位披着鲜艳的红色薄绉纱布沙丽的中年妇女， 抓起爬到她孩子背上的一只大蟑螂， 一下就仍到了站台下的铁轨之间。远处铁轨间一些衣衫褴褛的工人正在整修路基， 几十头瘦骨嶙嶙的小毛驴慢吞吞地将黄沙和水泥从远处驮来。印度各地的车站内外，常常看得见神牛、流浪狗、乌鸦和老鼠在月台和铁道上游走觅食。在世界的任何其它地方你绝对看不到人和动物， 甚至昆虫能够象在印度这样的和睦相处。

根据生命轮回的理论， 今天跑在你面前的一只老鼠， 有可能就是你半年前去世的奶奶。在去圣城瓦拉纳西的列车上，有人告诉了我这样一个小故事： 有一只大黄狗在一家寺院门外焦急地徘徊着， 另一只黑狗上前好奇地问道： “你在等什么?” 那只大黄狗咬牙切齿地说：“ 我在等院中的主持，他出来的话， 我得好好地教训教训他，当年我做和尚时他太欺负人了。” 黑狗悄悄地看了黄狗一眼, 慢条斯理地说：“我就是那个主持。” 所以不杀生和吃素，是大部分印度人奉行终生的原则， 而善待一切生物，也就意味着尊重自已所有已经去世的亲朋好友。

列车渐渐驶进瓦拉纳西的站台，车窗外飘进一股肉类烧烤产生的焦糊味。马克吐温曾说，瓦拉纳西比历史还老，比传统还悠久，比传说更为原始。公元前５００多年，佛祖曾在萨纳斯首次布道，那里距离瓦城仅１０公里。当时，瓦拉纳西是佛教的圣地，人口已达百万，中国唐代高僧玄奘历经千辛万苦，最终要到的极乐西天指的就是瓦拉纳西。大约５００年之后，瓦拉纳西成为印度教的重镇。印度教徒为人的四大乐趣：住瓦拉纳西；结交圣人；饮恒河水；敬湿婆神。其中的三项都要在瓦拉纳西才能实现。

瓦拉纳西紧靠印度的母亲河 C 恒河(Ganga River), 恒河如古老岁月般地流淌，而古老的习俗却并未随岁月流逝。印度教的信徒们生前，他们沿着瓦拉纳西石阶，下河沐浴，洗脱罪恶和痛苦。濒死，他们将尽力来到圣城，死得其所。死后，他们的肉体将在河旁火化，骨灰撒入恒河。那些没有福份死在恒河边的人，其遗属也必尽其所能，将尸体背到此地。死后能在恒河边焚化，将骨灰洒入河中，是每一个印度教徒的心愿。

不管白天黑夜，天晴下雨，恒河边总是架着一堆堆木材在焚烧尸体，有权有势的人用的是檀香木，贫困百姓烧的是廉价木材。升腾而起的股股黑烟周围， 挤满了死者的亲朋好友，没人喧哗，没人哭泣。对印度教的信徒来讲， 能有机会投胎转世未尝不是一件好事。

在印度旅行三个星期之后， 我在阿蘩(Amristar)离境，前往巴基斯坦。三个星期中印度的火车载着我爬山涉水， 走南闯北。除了每辆列车或多或少总要晚点之外， 算得上是安全可靠。也可能只是我命大福大， 总能万事逢凶化吉。实际上，安全事故在年久失修的印度铁路早已经成为家常便饭，这儿每年有300多次大小事故， 几乎可以摊到每天一次。其实只要想到印度的铁路居然可以同时拥有四种不同的轨距: 1.67米, 1米, 0.76米和0.6米， 大概任何人都会不寒而栗了。

近几年来，除了安全事故问题，极端主义和恐怖主义份子似乎置 ”不杀生”的古训于脑后, 越来越热衷于在铁路上做文章。远的不说， 只要看看以往发生的恐怖事件， 就一定会动摇你去印度坐火车旅行的念头。2006年3月份，印度教圣城瓦拉纳西的火车站和寺庙同时发生爆炸，造成21人死亡，62人受伤。7月11日，孟买城郊铁路发生连环爆炸，183人死亡，714人受伤，举世震惊。另外，活跃在印东部地区的极左武装“纳尔萨尔分子”也常袭击铁路和车站。曾经促进印民族、宗教和社会融合的铁路，渐渐变成了冲突和矛盾的焦点。安全问题已经成为印度铁路最脆弱的一根枕木。

从喜马拉雅山脚到印度洋边， 从恒河河畔到阿拉伯海海滨， 每天1万5千列列车仍要按时出发， 其中1万1千列是旅客列车， 没有任何东西可以吓阻勇敢，乐天的印度人， 特别是当你知道： 自己今世的生活，已经是由前世的所作所为决定了的， 还有什么事情可以让你感到不安呢?

想象一下: 在同一天里,  1万1千列客车, 乘载着1千3百万旅客, 在印度的大地上同时向着四面八方疾驶。特别是那些拖着无预定二等硬座车厢的普通旅客列车, 里里外外都挤满了人, 就连车头也不例外,  远远望去，就像一条表面爬满了蚂蚁的大蚯蚓在向前蠕动。面对如此壮观的场景,  大概没有人会不承认： 不管印度人聪明不聪明，印度人起码是世界上最特别的民族。

“生活在鱼缸里的鱼可能并不知道鱼缸是什么样子的， 一个旁观者可能知道整体感， 却没有生活在其中的真实感受。”  我又想到了大趋势作者约翰. 奈斯比特的话。
