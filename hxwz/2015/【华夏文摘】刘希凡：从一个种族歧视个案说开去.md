种族歧视行为在美国的活动空间逐渐萎缩，这是几十年来的总体趋势，是美国人权团体奋斗的成果，也是社会进步的一个标志。美国社会总体上同声谴责任何形式的种族歧视，但对种族歧视个案的处罚方式和力度，各方的立场很不相同。几乎每一个与种族歧视有关的事件或案件都有争议。

2015年3月8日媒体公布报道了一段网上的录像。这段10秒钟的录像显示俄克拉荷马大学学生在参加社交活动途中一起唱种族歧视歌曲。这项社交活动是SAE兄弟会组织的，事发地点是在兄弟会包租的五辆大客车中的一辆。所唱歌词是：

“There will never be a n—– SAE/There will never be a n—– SAE/You can hang ‘em from a tree, but it will never start with me/There will never be a n—– SAE.”

歌词中有“黑鬼”和“吊死他们”的语言。事发后，大学校长发表讲话谴责这些学生是“可耻的”的并在24小时内宣布停止SAE兄弟会在该大学校园内的活动，限期关闭SAE兄弟会在校内的公寓。然而，社会舆论没有平息，对这起种族歧视行为一片谴责之声。在舆论的影响下，校长在48小时内宣布了第二轮处罚：将两名在大客车上带头领唱的学生除名。至此，舆论相对平息，整个事件似乎进入了善后阶段。

事情远远没有结束。SAE兄弟会雇佣了著名律师Jones(他曾为1995年俄克拉荷马市大爆炸案的罪犯辩护)。Jones於3月13日发表声明，主要是三点：(1)一些SAE兄弟会成员的安全因此受到威胁，需要受到关注。(2)事实还有待澄清。这个事件的起因是9秒钟的录像和一辆车上的别学生。当时共有5辆大客车，每辆车上有二十多名学生。(3)SAE兄弟会此时无意进入法律程序。这个声明是低调的防守姿态。

3月13日同日，纽约某律师事务所的合伙人Dillion和Kalser在<>日报发表观点：“开除学生侵犯了学生的权利“。律师们指的是言论自由的权利。两位律师引用美国宪法第一修正案，称：“一所大学惩罚学生表达为他人所忌的观点，这不应该发生在一个自由社会里。”。这个声明有进攻的意思，起码也是蓄势待发。

至此时，被除名的两名学生分别公开道歉。学生的家长也公开道歉。

这个在我身边发生的事件有几个可圈可点之处：

1.	美国的社会舆论有巨大的影响力。宪法/刑法/民法在很多情况下只给出一个判定依据和刑罚范围。舆论对具体的惩处有相当的影响力。这是舆论自由的威力之一。因此，参与事件其中，用声明，信件，文章甚至Facebook表达意愿和倾向性意见是支持社会正义的重要形式。

2.	任何事情都不能一边倒。这两名被开除的学生收到了死亡威胁。对此，Jones的声明有保护当事人的意义。另外两名纽约律师的声明也是一家之言。美国宪法第一修正案本身只是一个概念性的界定，没有细则和细节，充满灰色地带。在灰色空间中，黑白双方可以打口水仗。

3.	退回50年，在1960年代，谁能想像这两名白人学生会因为一首种族歧视的歌曲而被一所中南部的大学除名？向前看50年，到2060年，这两个学生会受到何种处罚？这是个有意思的问题。任何权利，权益和尊重都是争取到的，都是动态的，没有一成不变的事，没有别人恩赐的好处。

4.	大学内和大学周边地带的华人在此事件中的表现以沉默为主。当我将我以校友名义给校长的信件以邮件方式发给近10位在大学内任教职的华裔朋友和几十名其他华裔朋友后，只有极少数人回复表示支持校长(在决定开除学生之前)加重处罚力度。如果时至今日还有人高喊“吊死黑人”，不要以为这同黄种人无关。

5.	如果有人在美国某个国家公园门口帖出告示：“华人与狗不得入内”，美国政府一定吃官司；如果有人在私家餐馆门口帖同样的告示，餐馆一定吃官司；如果有人在自己家门口帖同样的告示，外人可能看不见。如有人看见，并报告当地警局和媒体，会是什么结果？这是个有意思的问题，也是言论自由和种族歧视的有趣界定。

6.	种族歧视无处不在，无处不有，法律和舆论对其有限制和控制的功效，但不可能斩草除根。如果您在美国常住，有社会活动能力，那就必须争取自己权利。

我的一位律师朋友同我开玩笑说：如果有人在私家餐馆帖出了“华人与狗不得入内”的告示，不等华人团体告状，宠物协会也许先卷进来了。
