1

最近有机会参观了一所国内的高中。上课时是老师高声提问，同学们齐声回答。下课后同学们在操场上排成整齐的方队，在大喇叭的带领下做课间操。看着孩子们怯怯的表情，我一时有点走神，恍惚之间回到了多年前我自己做高中生时的情景。

那时我的日子很简单。整个学校的运转就像是方队齐步行进，我就是方队之中的一个新兵豆子，方队要走到什么地方去不是我该关心的事。我需要做的所有的事就是跟着大家走，脑子的唯一用途就是把考卷上的问题解决好。对于我和同学们来说，只有印在考卷上的那些问题才是问题，其他所有的都不是问题。为什么要读书？自己对什么感兴趣？将来想做什么样的事业？老师的话就一定要听吗？这些都不是问题。这些都是早就有了固定答案的，用不着我动脑子，也没有什么商量的余地。读书是为了上好大学：上了好大学，学校也荣耀，家长也荣耀，自己也有了金字招牌。个人兴趣要克制，因为总分才是硬道理。你将来想做什么？这得让你的总分来说话了。成绩够了，热门专业的大门就向你敞开。成绩不行，被冷僻专业收容，你这一辈子就玄了。老师的话就一定要听吗？脑袋里形成这么一个问句的那一刻就已经是罪恶了。

被这方队挟裹着，我总觉得有些不太对头，但我始终认为毛病是出在我这儿，且为此经常有种隐隐的自卑感。我从来没有怀疑过这方队有什么问题，因为从我记事起世界就是这样运行的：学生做操站的是方队，毛主席检阅的是方队，连党中央开会都那么一丝不苟整齐排列，俨然是坐着行进的方队。除了脑子一片空白地走在方队之中，嘴里大声重复着老师领喊的口令，我没有见过任何别的思维方式。那一年看洛杉矶奥运会入场式，那么多国家的选手在全世界数亿人的众目睽睽之下入场，居然步伐散漫、嘻嘻哈哈、举着相机到处拍照，让我很奇怪。我印象之中的运动员都该是齐步踏着《运动员进行曲》的节奏、目不斜视地排成方队入场的。

多年之后，我终于意识到人终归是要自己掌舵、自己体验世界的百味、自己变老、自己消逝在世界的视线之外的，所以也是要自己来回答那一串为什么的。我的生命属于我自己，不属于方队。我鼓足勇气从方队中溜号了。现在回过头来，看着方队之中怯怯的年轻人，几百上千个青春旺盛的十几岁的孩子，被指挥得那么整齐，如有被魔咒牵引一般，不免想起了平壤夜夜上演的巨型团体操。鸭绿江那边的方队规模更胜这边一筹。当年的希特勒也是方队的天才组织者。

2

但美国有些教育者极羡慕中国的同行能把那么多学生轻而易举地列成方队。他们经常发愁自己的学生自由散漫、令不行禁不止。美国不少政客也暗恨自己到处被掣肘，大大羡慕中国政府可以不必征求任何人的意见就能排出几百亿来修奥运场馆、铺高速铁路、把挡着拆迁推土机的钉子户连根拔起。看着奥巴马总统奋力践行自己上台前的许诺却屡屡受挫，支持阵营日渐缩水，三年间好像老了十几岁，不知道他是否在偷偷羡慕隔壁家的草坪更青翠可人。

美国的教育者和政客羡慕的中国特产叫做绝对权力，也就是人不受制约地号令他人的能力。俗话说物以稀为贵，绝对权力这东西在美国成不了太大的气候，所以有不少人想念它。尤其现在中国的经济奇迹成了美国人啧啧称道的热门话题，于是绝对权力也成了成功秘诀之一。

自然界有一种现象叫做自相似，指的是某些生物体和自然景观在从大到小的各种尺度上都显现出同一种形状特征。一个例子是树的分叉结构：树干上的分枝模式类似于大树枝上的分枝模式，也类似于小树枝上的分枝模式，这样的相似性一直延伸到每一片树叶上的叶脉结构。绝对权力在中国也是如此，其树状结构从上到下，从领袖到炙手可热的朝臣和大吏，直到百十户人家的村长。即使是无权无势的教师和平头百姓也拥有绝对权力 C 对孩子。孩子是中国权力金字塔的最底层承受者，一生下来就被来自所有各色人等的绝对权力操纵。孩子不仅不被允许有反驳父母指令的余地，连讨价还价都会让父母大为光火。我的父母在外是善良懦弱的人，但在内对孩子的铁腕控制却是没有折扣。他们的脑子里从来没有动过要听孩子任何一点不同意见的念头。这也怨不得他们 C 他们从小到大就是这样被教育的。

我自己在做了父亲后立志逆反祖辈对待孩子的方式，在从选择学校到才艺课程到正餐零食的各种大小事上给孩子自己做决定的机会。我给他们提供参考意见，但允许他们自己做出最后决定。我也允许孩子跟我讨价还价来争取更多的电子游戏时间。但我发现经常要极度克制才能给孩子让出一点发言权，因为以势压人的欲望总是源源涌出不可遏止。我每每懊悔于自己的简单粗暴，却每每旧疾重犯。这就是长期被绝对权力的空气潜移默化的结果。一个试图觉悟的个人尚且如此本性难改，那么十几亿浑然不觉的大众的惯性又该有多大呢。

3

孩子出生时是一张白纸，其后的近二十年中每时每刻都受着成年人的强力影响。这样的力量来刻写孩子的世界观，就像是重型坦克压过松软的土地。这坦克的两条履带就是言教与身教。言教是孩子耳朵里听到的父母与老师怎么说，身教是孩子眼睛里看到的父母与老师怎么做。身教胜于言教 C 不管拥有绝对权力的人嘴里谈论的是平等、爱心还是和谐，孩子领会到的只是绝对权力的天威。他们自小就习惯了那种压垮自己脊梁骨的力量在这个世界上的合法存在。在从成年人那里学到的所有教诲之中，给他们烙印最深的就是绝对权力的核心教诲：人与人是不平等的。我可以被有权力的人合法号令，也可以在自己获得权力后合法号令他人。

从小习惯了被绝对权力操纵，长大也就习惯了用绝对权力操纵别人。老师不允许学生发表自己的意见，学生在长大之后也就习惯了不允许掌握在自己手心之中的人发表意见。中国的绝对权力之树就这样历经数千年而不腐，并在本朝更发扬光大。至今记得多年前一位在加拿大的大陆留学生写的一句话：在加拿大有一种人：人；在中国有两种人：人上人和人下人。我想，中国与西方发达国家相比有政权结构、经济发展程度等诸多差别，这其中最基本的一个就是这个一针见血的观察。在国人的眼里，人上人和人下人并不是自己的同类。世界上这么多直立行走的动物，其中只有一个是人 C 那就是我自己。人上人是有权侵犯我的动物，人下人是我有权侵犯的动物。我在人上人面前要诚惶诚恐，在人下人面前要趾高气扬。我与别的直立行走动物之间只是利用和竞争的关系，并没有爱、关注、或救助的义务。我想从小悦悦身边若无其事走过的十八个路人大概就是持这样的世界观。中国枝繁叶茂的绝对权力大树就是被这种深入到人骨髓中的观念撑起来的。

人的思想观念决定了政权结构，政权结构又回过头来影响了人的思想观念，但这两者的地位是不对等的。人的思想观念是根，政权结构只是枝叶。如《阿Q正传》里写的那样，政权结构的改变对人的思想观念的影响是微弱的。民国的先驱们提倡的是“一种人”的平等思想，这与国人的“两种人”意识完全南辕北辙。虽然国民党为了适应中国的国情已经在实际操作中掺入了不少绝对权力的成分，但最终还是输给了共产党，因为共产党才是最适应中国国情的玩弄绝对权力的真正大师。我想，如果西方的选举制和三权分立被空降到三十年之前的中国，恐怕会严重水土不服、上吐下泻、落荒而逃。即使这种空降发生在今天，我还是不看好其生命力，因为在这三十年间国人的“两种人”世界观并没有什么变化。所以中国在文明之路上的进化不可能跳过普通中国人的思想观念的进化。要改变十几亿人的惯性的思想启蒙之路必然是漫长曲折，走在这拓荒之旅最前排的就是父母和教育者。

4

权力有两个要素：感召力和胁迫力。感召力是人上人对人下人精神的驾驭，出自于人下人的自愿，胁迫力则来自于人下人的恐惧感。要把几百上千人统摄在一个方队之中、要让数亿人都呼喊同一个口号，感召力与胁迫力缺一不可。毛泽东政权在成立之初有强大的胁迫力，也有一定的感召力。其时毛周的逝世尚可煽得民众的大把眼泪。几代之后感召力荡然无存，政治家们就没有这样的哀荣了。如果社会结构仅靠维稳之类的胁迫力维持，那么火药桶的爆炸就只等着新一代兼具感召力和胁迫力的挑战者来划那一根火柴了。这样的轮回已经在中国上演了很多次。

这轮回发生在国家民族的大尺度，也发生在家庭的小尺度。在孩子小的时候，父母的绝对权力尚有感召力来维持。等到了青春期，孩子的独立意识与逆反心理最终如燎原烈火，把父母的感召力一扫而光，只剩下靠利益维系的胁迫力。这时父母的任何指令都已经无法引起孩子的触动，进一步的胁迫又只能招致孩子进一步的逆反。有道是“哪里有胁迫，哪里就有反抗”。这场战役硝烟散尽之后，孩子学到的教训就是：这个世界是我的敌人，只有我奋斗得来的绝对权力才是征服这个世界的唯一武器。他们长大之后便顺理成章地拿起这武器，开始对下一代的胁迫。

青春期孩子对父母的逆反与中国百姓对当权者的逆反都是出于一个原因：人终归是要靠自己的判断、不是靠别人的押解来走路的。康德的墓碑上刻有一句广为人知的话：“有两样东西，愈持久地加以思索，就愈使心灵充满始终新鲜不断增长的景仰和敬畏：在我之上的星空和居我心中的道德法则。”我读这句话时注意到的是“居我心中”这四个字：人的行事准则是本来就居我心中的内在标准，不是被别人强加的外在标准。这是与万有引力定律一样基本的法则。懂得了万有引力定律，人类就可以造出宇宙飞船九天揽月，不懂得这样的基本法则，中国人就只能代代承受流血千里的权力轮回了。

如果父母和教育者能放下权力欲、鼓励孩子发现他们自己心中的道德法则、尊重孩子的选择，不因孩子的自行其是而恼羞成怒，这就是爱了。绝对权力的基石是我与别人的不平等，爱的基石是我与别人的平等 C 我的意愿与别人的意愿同样合情、同样掺杂着私欲与偏见、同样合法存在、同样值得被尊重。我珍视我的意愿，所以我珍视别人的意愿。

孩子感受到了爱，他们就学会了爱 C 这就是爱的身教。父母和教育者把孩子当成了平等的人，孩子也就把同类当作了平等的人，当作了值得他们奉献爱心和才智的对象。在爱中长大的孩子拥有内在的安全感，所以对他们最重要的事就不再是排好名次、上好大学、找好工作、挣大钱 C 这些是只对懦夫有吸引力的外在安全感。他们感兴趣的是自己去摸爬滚打、跌跌撞撞、在未知的世界中探索、体验新鲜的每一天。这样的生活比那些被人牵着鼻子走的方队成员要有意思多了。

温家宝曾问中国的教育者如何培养出大师级人才。温是康德的粉丝，可是如果他用康德的墓志铭来分析他的“大师之问”，他会发现他的问法是错的。大师不是被教育者培养出来的。教育者能做的唯一的事就是不绞尽脑汁扼杀未来的大师。他们只有两个选择：把孩子的活力窒息于方队之中，或是饶孩子一命、给孩子以自由呼吸的空间和自由探索的机会。如果他们选择了后者，大师就将从获得自由的孩子中间产生 C 他们不一定都有超一流的智商，但一定有最独立的判断、最强烈的好奇心和最持久的恒心。不过，这样的大师将只服从于“居我心中”的准则。他们见了来慰问的高官不会感激涕零，也不会做绝对权力的走卒或花瓶。他们倒极有可能成为颠覆绝对权力之歌利亚的大卫。我不知道这是不是温家宝期望见到的。

如果父母和教育者能有给孩子自由空间的爱心，大师将不过是他们小小的副产品而已。一代自由的个人将从他们那里诞生。有了自由的个人，中国向文明世界的进化就是水到渠成的事。
